## 第一章 引言

### 1.1 编写目的

本说明书旨在为"秀吧美业SaaS管理系统 V1.0"的用户提供详细的软件功能说明和操作指南，帮助用户快速掌握系统的使用方法，提高工作效率。

本说明书的预期读者包括：
- 美容院、美发店、美甲店等美业门店经营者
- 门店前台接待人员
- 门店服务技师
- 系统管理员
- 技术维护人员

### 1.2 项目背景

随着美业行业的快速发展，传统的手工记账和管理方式已经无法满足现代门店的运营需求。为了提高美业门店的管理效率、降低运营成本、提升客户体验，我们开发了"秀吧美业SaaS管理系统"。

本系统是一款专为美业门店设计的综合管理软件，提供收银管理、会员管理、预约管理、员工管理、库存管理、营销管理等核心功能，帮助美业门店实现数字化、智能化运营。

### 1.3 定义与缩写

| 术语 | 说明 |
|------|------|
| SaaS | Software as a Service，软件即服务 |
| SKU | Stock Keeping Unit，库存量单位 |
| API | Application Programming Interface，应用程序接口 |
| Vue | 渐进式JavaScript框架 |
| NestJS | 基于Node.js的企业级后端框架 |
| TypeORM | 对象关系映射工具 |

### 1.4 参考资料

## 第三章 软件安装

### 3.1 环境要求

在安装本系统之前，请确保您的计算机已安装以下软件：

1. **Node.js**：版本 16.0.0 及以上
2. **npm**：版本 7.0.0 及以上（随Node.js自动安装）

### 3.2 安装步骤

#### 3.2.1 下载软件包

从指定渠道获取"秀吧美业SaaS管理系统 V1.0"软件包，解压到本地目录。

#### 3.2.2 安装依赖

打开命令行工具，进入软件目录，执行以下命令：

```bash
# 进入后端目录
cd backend
npm install

# 进入前端目录
cd ../frontend
npm install
```

#### 3.2.3 初始化数据库

```bash
cd backend
npm run db:init
```

#### 3.2.4 启动服务

**启动后端服务：**
```bash
cd backend
npm run start:dev
```

**启动前端服务：**
```bash
cd frontend
npm run dev
```

#### 3.2.5 访问系统

打开浏览器，访问 http://localhost:5173 即可进入系统。

默认登录账号：
- 用户名：admin
- 密码：admin123

### 3.3 配置说明

#### 3.3.1 后端配置

后端配置文件位于 `backend/.env`，可配置以下参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| PORT | 服务端口 | 3000 |
| DB_TYPE | 数据库类型 | better-sqlite3 |
| DB_DATABASE | 数据库路径 | ./data/meiye_saas.db |
| JWT_SECRET | JWT密钥 | 随机生成 |

#### 3.3.2 前端配置

前端配置文件位于 `frontend/.env`，可配置以下参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| VITE_API_URL | 后端API地址 | http://localhost:3000 |

## 第五章 操作说明

### 5.1 收银管理操作

#### 5.1.1 开单收银

1. 点击左侧菜单"收银台"
2. 选择会员或散客
3. 点击"添加服务"或"添加商品"
4. 选择服务技师
5. 选择支付方式
6. 如需使用优惠券，点击"使用优惠券"
7. 点击"确认支付"完成收银

#### 5.1.2 会员卡充值

1. 在收银台选择会员
2. 点击"会员卡充值"
3. 选择充值套餐或输入充值金额
4. 选择支付方式
5. 点击"确认充值"

#### 5.1.3 查询订单

1. 点击左侧菜单"单据查询"
2. 设置查询条件（日期、状态、会员等）
3. 点击"查询"按钮
4. 在订单列表中查看订单
5. 点击订单编号查看详情

### 5.2 会员管理操作

#### 5.2.1 添加会员

1. 点击左侧菜单"会员管理"
2. 点击"添加会员"按钮
3. 填写会员基本信息
4. 上传会员照片
5. 设置会员标签
6. 点击"保存"完成添加

#### 5.2.2 会员充值

1. 在会员列表中找到目标会员
2. 点击"充值"按钮
3. 选择充值套餐或输入金额
4. 选择支付方式
5. 点击"确认充值"

#### 5.2.3 管理会员标签

1. 点击"会员标签"
2. 点击"添加标签"
3. 输入标签名称
4. 点击"保存"

### 5.3 预约管理操作

2. 点击"新建预约"
3. 选择会员或录入散客信息
4. 选择预约日期
5. 选择预约时段
6. 选择服务项目
7. 选择服务技师
8. 点击"保存预约"

#### 5.3.2 预约配置

1. 点击"预约配置"
2. 设置营业时段
3. 设置预约规则
4. 点击"保存"

### 5.4 员工管理操作

#### 5.4.1 添加员工

1. 点击左侧菜单"员工管理"
2. 点击"添加员工"
3. 填写员工基本信息
4. 设置员工职位
5. 设置员工服务项目
6. 点击"保存"

#### 5.4.2 员工排班

1. 点击"排班管理"
2. 选择排班日期范围
3. 为员工设置班次
4. 点击"保存排班"

#### 5.4.3 考勤管理

1. 点击"考勤管理"

#### 5.5.1 添加商品

2. 点击"添加商品"
3. 填写商品信息
4. 设置商品分类
5. 设置商品规格

## 第六章 常见问题

### 6.1 安装问题

**Q1：npm install 报错怎么办？**

A：请检查Node.js版本是否满足要求（v16.0.0及以上），尝试使用淘宝镜像：
```bash
npm config set registry https://registry.npmmirror.com
```

**Q2：启动后端服务报端口被占用？**

A：修改 `backend/.env` 中的端口号，或关闭占用端口的程序。

**Q3：数据库初始化失败？**

A：请确保 `backend/data` 目录存在且有写入权限。

### 6.2 使用问题

**Q1：登录后页面空白？**

A：请检查后端服务是否正常启动，打开浏览器控制台查看错误信息。

**Q2：数据无法保存？**

A：请检查数据库文件是否有写入权限，确保 `backend/data/meiye_saas.db` 文件存在。

**Q3：图片上传失败？**

A：请检查服务器磁盘空间是否充足，上传目录是否有写入权限。

**Q4：预约时段显示已满？**

A：请检查预约配置中的时段设置和员工排班情况。

**Q5：收银时找不到商品？**

A：请检查商品是否已上架，库存是否充足。

### 6.3 其他问题

**Q1：如何备份数据？**

A：复制 `backend/data/meiye_saas.db` 文件即可完成数据备份，也可在系统中使用数据备份功能。

**Q2：如何修改密码？**

A：进入"个人中心" → "修改密码"。

**Q3：如何添加新用户？**

A：使用管理员账号登录，进入"员工管理" → "添加员工"，然后在用户管理中设置账号。

**Q4：系统运行缓慢？**

A：请检查服务器性能，清理临时文件，优化数据库。
