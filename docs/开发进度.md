## API接口列表

### 小程序API (新增)
| 接口 | 方法 | 说明 |
|------|------|------|
| /api/miniapp/login | POST | 微信登录 |
| /api/miniapp/bind-phone | POST | 绑定手机号 |
| /api/miniapp/member/info | GET | 会员信息 |
| /api/miniapp/member/balance | GET | 余额余次 |
| /api/miniapp/member/consumptions | GET | 消费历史 |
| /api/miniapp/appointment | POST | 创建预约 |
| /api/miniapp/appointments | GET | 预约列表 |
| /api/miniapp/recharge | POST | 在线充值 |
| /api/miniapp/shop/info | GET | 商家信息 |
| /api/miniapp/team | GET | 团队展示 |
| /api/miniapp/promotions | GET | 优惠活动 |

### 报表API (新增)
| 接口 | 方法 | 说明 |
|------|------|------|
| /api/report/perm-dye-conversion | GET | 烫染转化率 |
| /api/report/customer-flow | GET | 客流趋势 |
| /api/report/customer-retention | GET | 客户留存 |

### 系统管理API (新增)
| 接口 | 方法 | 说明 |
|------|------|------|
| /api/operation-logs | GET | 操作日志列表 |
| /api/operation-logs/stats/* | GET | 日志统计 |
| /api/system-config | GET/POST | 系统配置 |
| /api/backup/export/* | POST | 导出备份 |
| /api/backup/list | GET | 备份列表 |
| /api/backup/restore/:filename | POST | 恢复备份 |

## 启动说明

### 后端启动
```bash
cd backend
npm install
npm run start:dev
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

### 小程序启动
```bash
cd miniprogram
npm install
npm run dev:mp-weixin
# 使用微信开发者工具打开 dist/dev/mp-weixin 目录
```
